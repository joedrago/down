If developing on android, this directory should contain the following:

* android/sdk: "sdk" directory from the ADT Bundle
  * http://developer.android.com/sdk/index.html
  * Run the SDK Manager once, check Tools and API 8, then Install Packages

* android/ndk: "ndk" dir from Android NDK
  * http://developer.android.com/tools/sdk/ndk/index.html

* ant: ant tool binary release (should have a /bin subdir)
  * http://ant.apache.org/bindownload.cgi
